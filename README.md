# Softball Stats

A Streamlit app for exploring softball statistics.

## Tabs
- **Batting Order** — lineup and order analysis
- **Season Stats** — current season aggregations
- **Career Stats** — multi-season career totals
- **Player Profile** — deep dive on individual players

## Setup
\`\`\`bash
python3 -m venv venv
source venv/bin/activate
pip install -r requirements.txt
streamlit run app.py
\`\`\`
EOF
